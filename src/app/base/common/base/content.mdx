
# 点和面

## 顶点 Vertex

顶点，就是构成图形的点，比如一个三角形有三个顶点，一个四边形有四个顶点，一个立方体有八个顶点

## 面 Polygon

面是由多个顶点组成的多边形。通常在3D模型中，一个面由3个顶点组成，也称为三角形（Triangle）。这意味着大多数3D模型由大量三角形构成。

## 顶点与shader的关联关系

顶点着色器（Vertex Shader）负责处理顶点的位置和属性，并将它们传递给片段着色器（Fragment Shader）。顶点着色器通常使用顶点缓冲区（Vertex Buffer）来存储顶点的数据，这些数据包括顶点的位置、颜色、法线等属性。在顶点着色器中，我们可以使用顶点缓冲区中的数据来计算顶点的位置和属性。例如，我们可以使用顶点的位置数据来计算顶点的世界坐标，使用顶点的法线数据来计算顶点的光照效果等。

在shader 中，有多少个顶点 就调用多少次vert函数


## 面和shader的关联关系

面和shader 没有直接关系，但是将面栅格化后像素，像素和frag函数有关系，有多少像素就调用多少次frag
